SIT

TC59 - User has placed one order

  1. User must be logged in
  2. User is logged on desktop device
  3. Searched item should exist (example Ananas)

Test steps

  1. Enter the "Ananas" in search bar
  2. Click on "Ananas" item from dropdown menu if it is there or go to link "svi proizvodi" from dropdown menu to see all items that contain "Ananas" name
  3. Click on the chosen product
  4. Click on "Dodaj u korpu"
  5. Click on "Pregledaj korpu"
  6. Click on "Izaberi način dostave"
  7. Choose a payment method
  8. Click on "Potvrdi plaćanje" to confirm the payment

Expected result

User has placed one order. User sees a message that the order has been successfully completed.
